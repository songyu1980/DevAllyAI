# DevAlly AI

[![GitHub](https://img.shields.io/badge/GitHub-songyu1980-blue?logo=github)](https://github.com/songyu1980)
[![Hugging Face Space](https://img.shields.io/badge/%F0%9F%A4%97%20Hugging%20Face-Spaces-yellow)](https://huggingface.co/spaces/songyugod/devally-ai)

**DevAlly AI** 是一款由 **songyu** 开发的专为开发者设计的 AI 编程助手。

## 🌟 项目简介
与普通的聊天 AI 不同，DevAlly AI 旨在深度融入开发者的工作流。它不仅能提供代码建议，还能：
- **深度理解项目结构**：不再局限于单段代码，而是理解整个项目的上下文。
- **自动化执行文件操作**：直接辅助完成文件读写与修改任务。
- **可视化代码差异（Diff）**：提供直观的代码变更对比，确保每一行改动都清晰可见。
